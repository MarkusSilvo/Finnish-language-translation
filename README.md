Extract the .zip file  
Send .xml file to AdminCP for your site. Appearance -> Languages -> Import

- Question Threads
- User Activity by Xon
- Redis cache
- [TH] UI.X
- [TH] Nodes
- [TH] Last Post Avatars
- [TH] Holidays
